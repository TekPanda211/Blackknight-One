\# Welcome



Welcome to \*\*Blackknight One\*\*.



Blackknight One is an open-source \*\*Enterprise Identity Engineering Platform\*\* built to help organizations discover, understand, validate, and continuously improve Microsoft Entra environments.



Unlike traditional reporting tools that simply collect Microsoft Graph information, Blackknight One correlates identity, authentication, authorization, governance, operational health, and Infrastructure as Code into a unified operational experience.



Our mission is simple:



> \*\*One Source of Truth for Enterprise Identity Engineering.\*\*

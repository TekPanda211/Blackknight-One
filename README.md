\# Terraform Entra Lab



Infrastructure as Code lab focused on Microsoft Entra ID, Azure, Identity Governance, Conditional Access, Access Packages, group management, and IAM security auditing.



\## Project Goals



\- Build Microsoft Entra ID identity resources with Terraform

\- Document Access Package and Conditional Access design

\- Use Microsoft Graph and PowerShell for IAM security checks

\- Support audit readiness through repeatable validation

\- Practice Git, GitHub, Terraform, and Infrastructure as Code workflows



\## Repository Structure



\- `docs/` - IAM design and audit documentation

\- `terraform/` - Terraform configuration files

\- `scripts/` - Microsoft Graph, PowerShell, and Azure CLI audit scripts

\- `examples/` - Example identity engineering scenarios

\- `modules/` - Future reusable Terraform modules

\- `images/` - Diagrams and screenshots

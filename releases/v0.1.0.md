\# Terraform Entra Lab v0.1.0



This release establishes the initial Microsoft Entra ID Infrastructure as Code foundation for the project.



\## Features



\- Terraform project initialized

\- Microsoft Entra ID (AzureAD) provider configured

\- Azure CLI authentication

\- Modular Terraform configuration

\- Department-based security groups

\- Dynamic group creation using `for\_each`

\- Standardized naming conventions

\- GitHub repository structure

\- Initial engineering documentation



\## Infrastructure Created



\- 27 Microsoft Entra ID security groups



Departments include:



\- IT

\- Engineering

\- Security

\- HR

\- Finance

\- Sales

\- Marketing

\- Support

\- Contractors



Each department contains:



\- Users

\- Admins

\- Readers
